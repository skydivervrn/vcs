You are given the following structs declarations in a Go program:

type Engine struct {
	Type  string
	Power int   
	Turbo bool
}

type Car struct {
	Make   string
	Model  string
	Year   int 
    Price  float32
	Engine Engine
}

Your task is to serialize a Car struct instance, for example:

engine := Engine{
    Type: "Inline-4",
    Power: 300,
}

car := Car{
    Make: "Tesla",
    Model: "model 3",
    Year: 2022,
    Engine: engine,
    Price: 49999.99,
}

To produce the following YAML output in the format:

make: Tesla
model: model3
year: 2022
engine_type: Inline-4
max_horsepower: 300
turbo: false

Your only objective is to add the correct YAML struct tags to the Car and Engine struct fields to obtain the specific YAML format/output described above.

Tip: Remember to use the correct struct tags to rename fields and directly embed nested struct fields into the parent structure with inline. To exclude non-required fields from the output, use - for complete omission and omitempty for conditional exclusion based on the field's zero value.


Sample Input:
Tesla model3 2022 49999.99 Inline-4 300 false

Sample Output:
make: Tesla
model: model3
year: 2022
engine_type: Inline-4
max_horsepower: 300
turbo: false
