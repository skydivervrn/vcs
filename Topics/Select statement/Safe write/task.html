As you may know, a Go program terminates with an error when attempting to write data to a channel that already has data.

Your task is to write a safeWrite() function for the dataChan channel. The function returns a bool data type. If the write is successful, it returns true; otherwise, it returns false. Most importantly, the program continues to execute even when the channel is blocked.

This problem uses the hidden check() function to check if you correctly implemented the code of the safeWrite() function.
