Write a Go program that starts a timer and reads data from a channel (the data gets written from a concurrent process). If reading is not possible immediately, the program waits.

However, once the reading is successful, the timer is reset to 1 second. If another read is successful within this time, the timer restarts. Finally, if the timer finishes and no data has been read, the program terminates and prints the message "exit".

The algorithm for the task is as follows:

  • The program starts a timer as soon as it begins.

  • The read branch gets executed when data can be read from the channel. In this branch, you must print the data and reset the timer.

  • If another read is successful within the timer's duration, the timer resets again.

  • If the timer finishes and no more data can be read from the channel, the timer's branch is triggered, leading to program termination.

Remember that the program should print the message "exit" upon terminating!

This problem uses the hidden check() function to check your solution for this coding task.


Memory limit: 512 MB
Time limit: 15 seconds
