Write a Go program that will continuously read information from the console and write it to a random channel. The channels for writing and inputting data are already defined in the program. Your task is to organize the writing of received data to the channels.

Tip. You can leave the body of the case expression empty, the important thing is the process of writing to the channel.

This problem uses the hidden check() function to check your solution for this coding task.


Memory limit: 256 MB
Time limit: 8 seconds
